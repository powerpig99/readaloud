# Sample Document

This is a sample document to test the ReadAloud application.

## Introduction

ReadAloud converts text documents into natural-sounding speech using the latest open-source text-to-speech technology. Simply upload your markdown file, optionally provide a voice reference for cloning, and listen to your content.

## Features

The application supports multiple languages, voice cloning from short audio samples, and playback speed adjustment from 0.5x to 3x speed.
